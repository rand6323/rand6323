### Python

#### gmail-email-sender
GoogleのAPIを何か使ってみようと思い作成\
Googleカレンダーに自動で登録を行うコードも書いたのでそのうち上げたい

#### file_organizer
ファイル整理を行うときに手作業だと時間がかかることもある作業を自動化できるよう作成\

#### face_recognition

#### image-optimize

#### ImgCompressor_kivy-android
